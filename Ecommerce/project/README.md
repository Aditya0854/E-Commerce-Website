# Ecommerce Project

## Description
This project is an Ecommerce website that allows users to browse and purchase products online.

## Features
- User authentication
- Product listings
- Shopping cart functionality
- Responsive design

## Installation
To install the project, run the following command:
```
npm install
```

## Usage
To start the server, run:
```
node server.js
```
